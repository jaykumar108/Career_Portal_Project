Career Portal API Documentation

Authentication

Base URL: http://localhost:5000/api

Most endpoints require authentication using JWT token in the Authorization header:

Authorization: Bearer <your_token>

Test Token Examples:

User Token: eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJ1c2VySWQiOiIxMjM0NTYiLCJyb2xlIjoidXNlciJ9.example_user_token

User Routes

POST /api/users/signup

Register a new user

Request Body:

{
    "name": "John Doe",
    "email": "john.doe@example.com",
    "password": "SecurePass123!",
    "phone": "+91 9876543210",
    "address": "123 Main Street, Mumbai, India"
}

Response (201 Created):

{
    "success": true,
    "message": "User registered successfully",
    "data": {
        "userId": "123456",
        "name": "John Doe",
        "email": "john.doe@example.com",
        "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..."
    }
}
